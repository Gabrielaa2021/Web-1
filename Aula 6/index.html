Flex Box

São caixas flexíveis que podem se adaptar aos breakpoints declarados no CSS

Elementos e atributos trabalhados junto com o Flex Box:

      Display: exibe um elemento
      Flex-direction: direção dos elementos, por padrão declaramos ele em row (linha)
      Flex-wrap: define quebra de linha da caixa
      Flex-float: atalho para o uso do direction e wrap
      justify-content: conteúdo justificado
      align-intems: alinhamento de intens dentro do elemento
